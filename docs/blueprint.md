# **App Name**: Kite

## Core Features:

- User Authentication & Roles: Authentication and role-based access control for different user types (Owner, Admin, PM, Sales, Member) within an organization. Users can switch between organizations if they are members of multiple ones.
- CRM Management: CRM functionality for managing Accounts and Opportunities. Sales users can create and manage accounts and opportunities. A 'Convert to Project' action is available upon winning an opportunity.
- Project Management: Project Management features including project listing, overview, WBS tree, allocations, risks/RAID, changes, and reports. PMs can create and modify projects and WBS, as well as manage allocations.
- Time-sheeting & Expenses: Time-sheeting and Expense submission interface for members. PMs have access to timesheet approval functionality.
- Rate Cards & Calendars: Rate card and calendar management for defining bill/cost rates and work schedules/holidays.
- Reporting & Analytics: Portfolio view with CPI/SPI heatmaps and drill-down to WBS for comprehensive project reporting.
- Project Optimization Assistant: AI tool that provides suggestions for optimizing project allocations based on member skills and availability.

## Style Guidelines:

- Primary color: #3498DB (RGB) – A vibrant blue to convey trust and professionalism, aligning with the CRM and project management aspects.
- Background color: #ECF0F1 (RGB) – A light gray to provide a clean and neutral backdrop, ensuring readability and focus on content.
- Accent color: #E74C3C (RGB) – A striking red used sparingly for critical actions and alerts, drawing user attention to important information.
- Headline font: 'Poppins' (sans-serif) for headlines, offering a modern and clean look.
- Body font: 'PT Sans' (sans-serif) for body text, ensuring readability and a balanced typographic structure.
- Use a consistent set of icons throughout the app to represent different actions, modules, and statuses. Icons should be simple and easily recognizable.
- The app will use a clear, intuitive layout with a sidebar for navigation and a main content area. Utilize grids and spacing effectively to create a balanced and visually appealing interface.
